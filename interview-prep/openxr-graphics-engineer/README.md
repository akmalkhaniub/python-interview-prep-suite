## 🎯 Key Focus Areas (Based on JD)
- **Standardization:** Mastering the OpenXR spec for cross-vendor support.
- **Runtime Depth:** Understanding how headsets communicate with the OS.
- **Latency:** Minimizing "Motion-to-Photon" latency for comfort.
- **Shaders:** Designing efficient visual effects for stereoscopic displays.
- **Hardware Abstraction:** Writing code that scales from mobile (Quest) to PC (SteamVR).
