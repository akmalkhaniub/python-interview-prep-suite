# Software Engineer III - OpenXR Developer

## Role Overview
**Role:** Software Engineer III (OpenXR)
**Compensation:** $130,000 – $200,000 (USD equivalent)
**Location:** Remote
**Type:** Full-time

Build cross-platform XR experiences using OpenXR expertise. Combine deep graphics and runtime engineering with XR-specific knowledge to ship experiences that work seamlessly across headsets (Quest, Vision, SteamVR, etc.) and platforms.

## Key Responsibilities
- **XR Development:** Build and maintain XR applications using OpenXR and adjacent SDKs.
- **Runtime Engineering:** Implement rendering, input, and runtime integration for headsets.
- **Optimization:** Optimize performance, frame timing, and latency for XR workloads.
- **Cross-Platform:** Build abstractions across Quest, Vision Pro, SteamVR, and others.
- **Architecture:** Conduct code reviews and contribute to architectural decisions.
- **Interaction Design:** Partner with designers on XR interaction patterns.

## Required Skills & Qualifications
- **Experience:** 4+ years in production software, specifically XR or graphics.
- **Languages/APIs:** Strong C++ and modern graphics APIs (Vulkan, DirectX, OpenGL).
- **XR Mastery:** Hands-on experience with OpenXR or comparable XR runtimes.
- **Engines:** Familiarity with Unity or Unreal Engine is a plus.
- **Graphics Deep Dive:** Experience with shaders, rendering pipelines, and optimization.
- **Fundamentals:** Solid software engineering principles.

## Behavioral Traits
- Curiosity to dig into runtime systems and graphics pipelines.
- Strong written communication for technical decision-making.
- Test-and-learn mindset (ship fast, measure, iterate).
- Comfort working asynchronously across time zones.
