# Senior Azure LLMOps Engineer - Job Description

## About the job
We are building an enterprise-scale, Azure-native Document Intelligence and Retrieval-Augmented Generation (RAG) platform. We are seeking a highly skilled MLOps Engineer to operationalize, scale, and govern ML/LLM pipelines across this ecosystem.

## Responsibilities
* **ML/LLM Pipeline Operationalization**: Productionize pipelines for data ingestion (Graph API), classification, extraction, enrichment, and retrieval using Azure Data Factory and Azure Functions.
* **LLMOps & RAG Management**: Deploy and optimize RAG pipelines using Azure OpenAI, LangChain, and Azure AI Search.
* **Lifecycle Management**: Implement CI/CD for prompt and configuration changes using Azure ML and GitHub Actions/Azure DevOps.
* **Reliability**: Manage schema drift, data validation, and metadata consistency across SharePoint, Blob Storage, and Cosmos DB.
* **Monitoring & Observability**: Build systems for tracking pipeline failures, latency, token usage, and cost using Azure Monitor and Application Insights.
* **Security & Governance**: Enforce RBAC, VNet isolation, and PII redaction. Implement safe AI practices and guardrails.
* **Performance Optimization**: Optimize prompt efficiency, caching, autoscaling, and tune chunk sizes/hybrid search weights.
* **Collaboration**: Work with Data Engineers and AI Engineers to provide reusable MLOps templates.

## Qualifications
* 6+ years of experience in MLOps, ML Engineering, or Platform Engineering.
* Strong expertise in the Microsoft Azure ecosystem.
* Proficiency in Python (pipelines, orchestration, APIs).
* Hands-on experience with CI/CD for ML systems and Containerization (Docker, K8s).

## Preferred (LLM / RAG Experience)
* Experience with Azure OpenAI Service, LangChain, and Vector DBs.
* Understanding of prompt engineering lifecycle and RAG evaluation techniques.

## Requirements added by the job poster
* Working in an onsite setting.
* 3+ years of work experience with Azure OpenAI.
* 4+ years of work experience with MLOps.
* 4+ years of work experience with Vector Databases.
