# Job Description: Software Engineer — Mobile (Kotlin Multiplatform)

## Company: ImagineArt
**Location:** On-Site — H-12, NSTP, Islamabad
**Status:** One of the fastest-growing GenAI companies ($35M+ ARR, 100M+ impressions)

---

## Job Summary
We are looking for a Kotlin Multiplatform Engineer who takes full ownership of what they build. This is not a ticket-based execution role. You will be responsible for shipping features end-to-end — from understanding the problem, designing architecture, implementing, and iterating post-release.

---

## What You'll Do
- **Own features end-to-end:** problem → architecture → implementation → release → iteration
- **Build and maintain shared modules:** Android and iOS using Kotlin Multiplatform (KMP)
- **Develop modern UI:** Compose Multiplatform (CMP)
- **Architectural decisions:** Focus on long-term scalability
- **Collaborate:** Work with product and design to shape features
- **Code Quality:** Write clean, modular, and testable code
- **Proactive fixing:** Identify issues in production and fix them with urgency

---

## What We're Looking For
- **3+ years of Android development** with strong Kotlin fundamentals
- **Hands-on experience with:**
    - Kotlin Multiplatform (KMP)
    - Compose Multiplatform (CMP)
- **Strong understanding of:**
    - Clean Architecture
    - MVI / state management patterns
    - Coroutines and asynchronous programming
    - Dependency Injection
- **Ability to write scalable, modular, and maintainable code**
- **Strong product mindset** — care about user experience
- **Ownership mentality** — proactive problem solver
