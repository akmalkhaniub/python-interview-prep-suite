# AI/ML Engineer Video Pitch Preparation

This module helps you prepare for the 1-2 minute introductory video requested by recruiters for AI/ML roles. It focuses on balancing your technical depth with communication clarity.

## 📂 Contents

1. [**01-Video-Pitch-Script.md**](./01-Video-Pitch-Script.md): A modular script template you can customize.
2. [**02-Technical-Highlight-Guide.md**](./02-Technical-Highlight-Guide.md): How to "verbalize" complex topics like RAG and Agents in seconds.
3. [**03-Recording-Checklist.md**](./03-Recording-Checklist.md): Environment, lighting, and delivery tips.

## 🚀 How to use this
1. Read the **Script Template**.
2. Fill in the bracketed `[ ]` sections with your specific projects (use the "Technical Highlight Guide" for inspiration).
3. Record a 90-second draft.
4. Review against the **Checklist**.
