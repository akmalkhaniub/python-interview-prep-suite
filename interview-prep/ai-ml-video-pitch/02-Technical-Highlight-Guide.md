# 02: Technical Highlight Guide

When recording a video, you don't have time for a whiteboard. You must use "Power Phrases" to signal expertise quickly.

## 1. Talking about RAG (Retrieval Augmented Generation)
**Don't just say:** "I built a RAG pipeline."
**Do say:** 
*   "I’ve implemented **Hybrid Search** (Vector + Keyword) to improve retrieval precision."
*   "I focus on **Query Transformation** and **Reranking** to handle complex user intents."
*   "I have experience with **Evaluation Frameworks** like RAGAS to quantify pipeline performance."

## 2. Talking about AI Agents
**Don't just say:** "I made an AI Agent."
**Do say:**
*   "I design **Multi-Agent Orchestrations** where specialized agents handle sub-tasks."
*   "I’ve implemented **Tool-Calling** and **Function-Calling** patterns to give LLMs real-world agency."
*   "I focus on **State Management** and **Memory Persistence** in long-running agentic loops."

## 3. Talking about Production/MLOps
**Don't just say:** "I put models in production."
**Do say:**
*   "I prioritize **Observability**, tracking token usage, latency, and drift in real-time."
*   "I build **Evaluation Benchmarks** to ensure new prompts don't cause regressions."
*   "I specialize in **Cost-Optimization**, using small-language models (SLMs) for simpler tasks to save on API costs."

## 4. Talking about Prompt Engineering
**Don't just say:** "I write good prompts."
**Do say:**
*   "I use **Few-Shot Prompting** and **Chain-of-Thought** reasoning to improve complex output accuracy."
*   "I implement **Structured Output** enforcement (JSON/Pydantic) for seamless backend integration."
