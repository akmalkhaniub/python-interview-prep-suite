# Senior Azure AI Solutions Engineer - Job Description

## About the job
We’re building a large-scale, enterprise-grade document intelligence and RAG platform on Azure. You’ll design and implement end-to-end AI pipelines from ingestion and classification to extraction, enrichment, indexing, and conversational consumption leveraging Azure AI Foundry as the core orchestration and experimentation layer.

## Responsibilities
* Architect and implement RAG pipelines using Azure OpenAI Service and Azure AI Foundry
* Build ingestion pipelines from SharePoint using Graph API and orchestrate with Azure Data Factory / Functions
* Develop document classification and extraction systems (OCR, layout parsing, entity extraction)
* Implement chunking, embedding generation, and vector indexing using Azure AI Search / Cosmos DB
* Design prompt templates, evaluation loops, and guardrails within Foundry
* Integrate LangChain-based routing and chaining for dynamic workflows
* Develop APIs (FastAPI) for downstream consumption (chatbots, copilots, REST endpoints)
* Implement evaluation frameworks (precision/recall, hallucination detection, grounding quality)
* Optimize cost, latency, and throughput across LLM workloads
* Ensure governance: RBAC, secure endpoints, audit logging, and data residency compliance

## Qualifications
* Strong hands-on experience with Azure AI Foundry (mandatory)
* Deep knowledge of RAG architectures and LLM lifecycle
* Experience with GPT-4 class models, embeddings, and prompt engineering
* Proficiency in Python (LangChain, orchestration frameworks)
* Experience with Azure services: Data Factory, Functions, Cosmos DB, AI Search
* Familiarity with OCR and document parsing (structured/unstructured data)
* Experience building production-grade APIs and pipelines

## Requirements added by the job poster
* Working in an onsite setting
* 3+ years of work experience with Azure Data Factory
* 4+ years of work experience with Azure AI Foundry
