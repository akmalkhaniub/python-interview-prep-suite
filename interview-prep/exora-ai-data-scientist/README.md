# Interview Prep: Data Scientist Engineer @ Exora AI

Exora AI is looking for a well-rounded ML Engineer who understands the **Foundational Mathematics** of ML as well as the **Software Engineering** required to deploy models at scale.

## Preparation Roadmap

### 1. Classical Machine Learning (The "Basics")
- [ ] **Supervised Learning:** Linear/Logistic Regression, Decision Trees, Random Forests, XGBoost/LightGBM.
- [ ] **Unsupervised Learning:** K-Means Clustering, PCA (Dimensionality Reduction), Anomaly Detection.
- [ ] **Model Optimization:** Hyperparameter tuning (GridSearch, RandomSearch, Bayesian), Regularization (L1/L2), and Cross-Validation.

### 2. Deep Learning (CNN & RNN)
- [ ] **Architectures:** CNNs for image/spatial data, RNNs/LSTMs for sequential data.
- [ ] **Frameworks:** TensorFlow vs. PyTorch (Be ready to discuss pros/cons).
- [ ] **Transfer Learning:** Using pre-trained models and fine-tuning them for specific tasks.

### 3. Data Engineering for Data Science
- [ ] **Preprocessing:** Handling missing data (Imputation), Outlier detection (Z-score, IQR), and Feature Scaling (Normalization vs. Standardization).
- [ ] **Feature Engineering:** Creating new features from raw data (encoding, binning, interaction terms).

### 4. Deployment & MLOps
- [ ] **Inference:** Batch vs. Real-time inference.
- [ ] **Monitoring:** Data Drift (distribution changes) and Model Drift (performance changes).
- [ ] **Documentation:** Using Git for model versioning and experiments.

### 5. Statistics & Problem Solving
- [ ] **Statistical Tests:** P-values, Hypothesis testing, Confidence intervals.
- [ ] **Metric Selection:** Accuracy, Precision, Recall, F1-Score, ROC-AUC, RMSE/MAE.
