# Python Game Developer

## Role Overview
**Role:** Python Game Developer
**Compensation:** $85,000 – $140,000 (USD equivalent)
**Location:** Remote
**Type:** Full-time

Build games, simulations, or game tooling in Python. Combine Python craft with games-specific work—shipping clean, maintainable systems that designers and engineers love to build on.

## Key Responsibilities
- **Game Features:** Build and maintain game features, tools, and backend systems.
- **Logic & AI:** Develop game logic, simulation systems, or AI behaviors.
- **Tooling:** Build internal tools and pipelines for designers and content creators.
- **Integration:** Integrate Python systems with engine, server, or web components.
- **Debugging:** Diagnose and resolve bugs across the game stack.
- **Mentorship:** Conduct code reviews and contribute to engineering practices.

## Required Skills & Qualifications
- **Experience:** 3+ years building production Python applications.
- **Technical Mastery:** Strong Python skills (async, optimization, packaging).
- **Frameworks:** Experience with Pygame, Panda3D, Ren'Py, or engine scripting (Blender/Unreal).
- **Domain Knowledge:** Familiarity with game backends, multiplayer systems, or simulation engines.
- **Fundamentals:** Solid software engineering and version control (Git).

## Behavioral Traits
- Curiosity to dig into complex systems and optimize them.
- Strong written communication for technical decision-making.
- Test-and-learn mindset (ship fast, measure, iterate).
- Comfort working asynchronously across time zones.
