# Senior Microsoft Fabric Data Engineer - Job Description

## About the job
Translate business requirements into scalable, production-ready solutions on Microsoft Fabric. Work directly with business and functional teams to understand processes, KPIs, data pain points, and translate them into actionable data models, pipelines, and Fabric assets.

## Responsibilities
* Hands-on ownership of end-to-end data architecture and implementation across Microsoft Fabric (OneLake, Lakehouse/Warehouse, Data Factory Pipelines & Dataflows Gen2, Notebooks/Spark).
* Design and build medallion architecture (Bronze/Silver/Gold) including schema design, delta tables, partition strategy, performance optimization, and conformed enterprise data models.
* Develop batch and near-real-time ingestion pipelines from databases, APIs, and file-based sources; implement CDC, SCD Type 1/2, audit columns, and upsert logic using Fabric Data Pipelines and Spark.
* Write production-quality SQL, PySpark, and Spark-based transformations in notebooks and pipelines; build reusable framework components and automation patterns.
* Implement data quality rules, validation logic, monitoring, observability, and SLAs, ensuring pipelines are reliable, scalable, and trustworthy.
* Troubleshoot and debug issues hands-on across ingestion, transformation, storage, and modeling layers; drive root-cause analysis and maintain playbooks/runbooks.
* Apply governance & security best practices, including RLS/CLS, sensitivity labels, lineage, data cataloging, and metadata management using Microsoft Purview.
* Set up and maintain CI/CD automation using Fabric Deployment Pipelines and Git; manage branching, release cycles, and environment promotion strategy.
* Optimize performance and cost by selecting the right Fabric compute modes (Warehouse vs Lakehouse), optimizing Delta Lake structures, partitioning, caching, indexing, and Spark tuning.

## Requirements
* 8+ years of hands-on experience in data engineering/data architecture, including 2+ years of practical implementation on Microsoft Fabric.
* Strong expertise in OneLake, Lakehouse/Warehouse architecture, Spark/Notebooks, Data Factory (Pipelines & Dataflows Gen2).
* Advanced SQL and PySpark skills with hands-on experience building notebook-based transformations, tuning Spark jobs, and working with Delta Lake.
* Proven experience designing and building medallion architectures, star/snowflake schemas, and conformed dimensional models.
* Strong understanding of CDC/SCD patterns and experience integrating operational/financial ERP sources like Sage, Xero, or similar systems.
* Experience implementing CI/CD with Git and Fabric Deployment Pipelines, including unit testing, validation checks, and automated promotion.
* Solid understanding of Azure backend components such as ADLS Gen2, Azure SQL, Azure Data Factory, Synapse, and migrations into Fabric.
* Hands-on experience with Microsoft Purview (data lineage, data catalog, sensitivity labels) and implementing GDPR-compliant data governance frameworks.
* Experience optimizing Microsoft Fabric capacity usage through workload isolation, capacity scaling strategies, query performance tuning, and cost monitoring.
* Strong Python/PySpark development capability with a deep understanding of orchestration patterns and pipeline resiliency.
* Good business domain understanding across finance, operations, and enterprise data.
