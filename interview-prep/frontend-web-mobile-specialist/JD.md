# Frontend Developer (Web & Mobile Specialist)

## Role Overview
**Role:** Frontend Developer
**Experience:** 3–7 Years
**Domain:** Technology / FinTech / Digital Platforms

Responsible for building high-performance, scalable, and visually refined user interfaces across both web and mobile applications. Translate complex requirements into intuitive, responsive, and user-friendly interfaces.

## Key Accountabilities
- **UI Implementation:** Translate UI/UX designs into high-quality production code.
- **Component Architecture:** Develop reusable, modular UI components.
- **Data Integration:** Integrate APIs (REST, GraphQL) and handle real-time data.
- **Performance:** Optimize for web and mobile (code splitting, lazy loading, caching).
- **State Management:** Use Redux, Context API, or Zustand for predictable data flows.
- **Mobile Development:** Maintain mobile apps using React Native, optimizing for gestures and navigation.
- **User Experience:** Improve usability, accessibility, and interaction quality.
- **Maintainability:** Write clean code and work within CI/CD pipelines.

## Required Skills & Qualifications
- **Core:** JavaScript (ES6+), HTML5, CSS3 (Flexbox/Grid), React.
- **Mobile:** React Native, mobile UI/UX patterns.
- **Advanced:** Performance optimization, asynchronous programming, data-intensive interfaces.
- **Soft Skills:** Problem-solving, attention to detail, clear communication.
- **Experience:** 3–7 years in frontend development with production-grade apps.

## Key Performance Indicators (KPIs)
- Performance and responsiveness (Web & Mobile).
- Code quality and maintainability.
- Implementation accuracy vs requirements.
- User experience quality.
