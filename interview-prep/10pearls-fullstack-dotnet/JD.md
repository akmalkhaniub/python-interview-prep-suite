# Job Description: Full Stack Developer (.NET, Angular, Azure) @ 10Pearls

## Role Overview
**Company:** 10Pearls (End-to-end digital innovation company)
**Status:** Experienced role focusing on architecting and developing scalable web applications using .NET, Angular, and Azure.

---

## Key Responsibilities
- **Development:** Build scalable applications using **.NET 8+** and **Angular** (preferred).
- **Architecture:** Design software solutions using **Clean Architecture** and **Microservices**.
- **Quality:** Ensure code quality through unit, integration, and automation testing.
- **Optimization:** Identify and implement performance improvements in existing software.
- **Cloud/DevOps:** Deploy and manage applications on **Azure**; handle **CI/CD** via Azure DevOps.
- **Mentorship:** Guide junior developers and foster knowledge sharing.
- **Collaboration:** Work in an **Agile/Scrum** environment with cross-functional teams.

---

## Technical Requirements
- **Experience:** 5+ years in **C# / .NET**; 2+ years in **API development**.
- **Frontend:** Proficiency in **Modern Angular** (preferred) or React.
- **Cloud Services:** Azure (App Service, Functions, Storage, Service Bus, AD).
- **Database:** Strong **MSSQL** skills (query optimization, database design).
- **Architecture:** Microservices, distributed systems, Docker/Kubernetes (preferred).
- **Testing:** Unit, integration, and automation testing methodologies.
- **Nice to have:** GraphQL.
