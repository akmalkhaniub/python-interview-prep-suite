# Interview Prep: Full Stack Developer @ 10Pearls

10Pearls is looking for a senior-level engineer who can lead development efforts using the **Microsoft Stack (.NET/Azure)** combined with **Modern Angular**.

## Preparation Roadmap

### 1. Modern .NET & C# (The "Backend" Pillar)
- [ ] **.NET 8+ Features:** Minimal APIs, Native AOT, Performance improvements.
- [ ] **Clean Architecture:** Domain-Driven Design (DDD), Repository Pattern, Dependency Injection.
- [ ] **Web API Best Practices:** Middleware, Action Filters, Rate Limiting, Versioning.

### 2. Modern Angular (The "Frontend" Pillar)
- [ ] **Standalone Components:** The new standard in Angular.
- [ ] **Signals:** Modern state management in Angular.
- [ ] **RxJS:** Mastery of Observables, Subjects, and Operators (switchMap, combineLatest).

### 3. Azure & DevOps (The "Cloud" Pillar)
- [ ] **Azure App Services & Functions:** Serverless vs. App-based hosting.
- [ ] **Service Bus:** Messaging for distributed systems/microservices.
- [ ] **CI/CD:** Azure DevOps pipelines (YAML based), Artifacts, and Releases.

### 4. Database & Microservices
- [ ] **MSSQL Optimization:** Execution plans, Indexing strategies (Clustered vs Non-clustered).
- [ ] **Microservices:** Service discovery, API Gateway, Circuit Breaker pattern.

### 5. Leadership & Testing
- [ ] **Mentoring:** Be ready to talk about how you've grown junior talent.
- [ ] **Testing:** TDD (Test Driven Development), xUnit, Moq, and Integration tests.
