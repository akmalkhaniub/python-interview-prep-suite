# Backend Engineer (Production Systems)

## Role Overview
**Role:** Backend Engineer
**Compensation:** $105,000 – $170,000 (USD equivalent)
**Location:** Remote
**Type:** Full-time

Design and build the services that power the product. Focus on APIs, data systems, and infrastructure choices that shape product capabilities. Deliver high-performance, reliable, and scalable backend solutions.

## Key Responsibilities
- **Service Design:** Build backend services, APIs, and data systems.
- **Data Modeling:** Design scalable data models and APIs.
- **Code Quality:** Write clean, well-tested code with solid architectural choices.
- **Optimization:** Diagnose and resolve performance and reliability issues.
- **Leadership:** Conduct code reviews and contribute to architectural decisions.
- **Collaboration:** Partner with frontend, infrastructure, and product teams.

## Required Skills & Qualifications
- **Experience:** 3+ years building production backend systems.
- **Languages:** Strong proficiency in Python, Go, Java, or Node.js.
- **Databases:** Solid understanding of relational (SQL) and non-relational (NoSQL) systems.
- **APIs:** Experience designing REST or GraphQL APIs.
- **Infrastructure:** Familiarity with cloud services and containerization (Docker, Kubernetes).
- **Fundamentals:** Strong system design and architectural principles.

## Behavioral Traits
- Curiosity to dig into systems and drive shipped improvements.
- Strong written communication and ability to explain technical decisions.
- Test-and-learn mindset (ship fast, measure, iterate).
- Comfort working asynchronously across time zones.
