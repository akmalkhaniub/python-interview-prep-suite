# Interview Prep: Lead AI Product Engineer

This role is a "dual-threat" position requiring deep technical expertise in **Full-stack SaaS** and **AI Orchestration**, combined with an **entrepreneurial/marketing mindset**.

## Preparation Roadmap

### 1. Full-Stack & SaaS Architecture
- [ ] **Next.js & Node.js:** SSR/ISR patterns, scaling strategies.
- [ ] **Stripe Integration:** Subscriptions, webhooks, usage-based billing logic.
- [ ] **Multi-tenancy:** Data isolation strategies in NoSQL (Firestore).

### 2. AI Orchestration (The "Core")
- [ ] **Claude API (Anthropic):** Working with system prompts, context management.
- [ ] **Structured Outputs:** Pydantic models for JSON enforcement in FastAPI.
- [ ] **Asynchronous Processing:** Handling long-running AI tasks (Celery/Redis or serverless).

### 3. Firebase Mastery
- [ ] **Security Rules:** Writing complex, non-trivial Firestore rules for B2B/B2C.
- [ ] **Real-time Sync:** When to use listeners vs. one-time fetches.

### 4. MarTech & Content Pipelines
- [ ] **Web Scraping:** Scraping for context (Playwright/BeautifulSoup).
- [ ] **CMS Integration:** Automated deployment to WordPress/Headless CMS.
- [ ] **SEO Automation:** Generating metadata, schema, and internal linking via AI.

### 5. Entrepreneurial Mindset
- [ ] **Speed vs. Quality:** Knowing where to cut corners to find PMF.
- [ ] **MRR Focus:** Understanding how code choices impact revenue share.
