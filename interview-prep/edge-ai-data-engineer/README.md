# Interview Prep: AI & Data Engineer @ Edge

Edge is looking for a "Product-minded Data & AI Engineer" who can bridge the gap between marketing analytics (GA4), core data pipelines, and agentic AI automation.

## Preparation Roadmap

### 1. GA4 & Marketing Analytics (The "Analytics" Pillar)
- [ ] **GA4 Event Schema:** Designing custom events and parameters.
- [ ] **GTM Server-Side:** Benefits for data privacy and performance.
- [ ] **BigQuery Export:** Querying nested GA4 data (event_params, user_properties).
- [ ] **Attribution Modeling:** First-touch, Last-touch, and Linear models.

### 2. Data Engineering (The "Infrastructure" Pillar)
- [ ] **ETL/ELT on GCP:** Using BigQuery as a Warehouse.
- [ ] **dbt (Data Build Tool):** Modeling data in the warehouse.
- [ ] **Orchestration:** Airflow or Dagster for pipeline reliability.

### 3. Agentic AI (The "Automation" Pillar)
- [ ] **CrewAI:** Multi-agent role-playing (Manager, Researcher, Sales Associate).
- [ ] **LangGraph:** Orchestrating complex business workflows with loops.
- [ ] **Tool Use:** Connecting agents to Salesforce/HubSpot via APIs.

### 4. CRM & Business Integrations
- [ ] **HubSpot/Salesforce APIs:** Managing contacts, deals, and custom objects.
- [ ] **Webhooks:** Handling real-time triggers (e.g., "New Lead" in HubSpot -> "AI Outreach" in Slack).

### 5. Compliance & Privacy
- [ ] **GDPR/CCPA:** Principles of data minimization.
- [ ] **Consent Mode v2:** How it impacts tracking and analytics.
