## 🎯 Key Focus Areas (Based on JD)
- **Traditional CV:** Beyond deep learning - filtering, morphology, and geometric calibration.
- **Feature Engineering:** Combining handcrafted features (color/texture) with DL embeddings.
- **Agentic AI:** Using LangChain/AutoGen to orchestrate vision tasks and tool-calling.
- **Model Deployment:** Optimization via ONNX and understanding edge constraints.
- **Multimodal:** Connecting visual inputs to LLM reasoning (VLMs).
